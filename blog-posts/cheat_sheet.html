VARIABLES AND ATTRIBUTES

attr_accessor basically creates getter and setter methods for you. This allows you to read and write what the variables are from outside the class. When creating a class, you can still initalize variables with whatever values you want. You can also initalize them to blank, and then use methods to set them later. see example

Class prac attr_accessor :variable def initalize(variable) @variable = variable end end test = Prac.new("hi") test.variable = "bye" variable now equals "bye" instead of "hi"
